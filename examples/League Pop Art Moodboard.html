The Direction

What we're going for

Roy Lichtenstein's comic-book vocabulary — flat saturated primaries, heavy uniform black outlines, Ben-Day dots, and a single explosive sound word — applied to the League site. Take League photos, halftone them, and hang an action burst off the side.

BAM!
ZAP!
Bold outlines. Three colors plus black and white. Nothing subtle.

Techniques

Dots, halftone & rays

Ben-Day = uniform dots, used for flat decorative fills. Halftone = variable-size dots, used for photo shadows and gradients. Starburst rays sit behind hero callouts. All three below are pure CSS — own-able, no licensing.

Starter Tokens

Drop into your CSS

The :root block powering this page. Copy it as the seed for the design system — colors, the heavy-outline scale, and the offset hard shadow that gives everything its cut-from-paper feel.

:root {
  /* Core ink + paper */
  --ink:            #0a0a0a;
  --paper:          #fdf6e3;
  --paper-cool:     #ffffff;

  /* Lichtenstein primaries (flat, saturated) */
  --pop-red:        #e8112d;
  --pop-yellow:     #ffd000;
  --pop-blue:       #1066d6;
  --pop-cyan:       #18b8d6;

  /* League bridge + secondaries */
  --league-orange:  #f5821f;
  --pop-pink:       #ff4f8b;

  /* Line + shadow language */
  --outline:        4px;   /* default heavy outline */
  --outline-xl:     6px;   /* hero outline */
  --hard-shadow:    8px 8px 0 var(--ink);  /* offset, no blur */
}
Rules of the Style

What makes it read as pop art

  • Thick uniform black outlines on every element — cards, buttons, images, bubbles. This is the single most defining trait.
  • Flat fills only. No soft gradients, no drop-shadow blur. Shadows are hard, offset, and solid black.
  • 3–4 colors max per composition, plus ink and paper. Saturated, high-contrast pairings.
  • One sound word per moment — POW, BAM, ZAP — in a burst, never two competing in the same view.
  • Halftone the photos, don't leave them photographic. Variable dots for skin and sky.
  • Display type is heavy and tilted — Arial Black / Impact, slight rotation, ink stroke around the fill.
  • Caption boxes — terse yellow narration panels with a black border, lifted straight from comic panels.
